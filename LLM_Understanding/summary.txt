conditional_edgs
======================

In Python, the with keyword is used for resource management.

Your code:

with MongoDBSaver.from_conn_string(DB_URI) as checkpointer:

means:

Create resource
↓
Use resource safely
↓
Automatically clean up resource
What Happens Internally

This:

with MongoDBSaver.from_conn_string(DB_URI) as checkpointer:
    ...

is roughly equivalent to:

checkpointer = MongoDBSaver.from_conn_string(DB_URI)

try:
    ...
finally:
    checkpointer.close()

So Python automatically handles cleanup.
